# NexTASK

AI-Powered Full-Stack Task Management Platform

🚀 Live Demo: https://nextask-frontend-kappa.vercel.app

📂 Repository: https://github.com/Sujit-S3/NexTASK

## Overview

NexTASK is a modern AI-powered task management and collaboration platform built using the MERN stack. It enables teams to create, assign, manage, and track tasks while providing real-time updates, notifications, analytics, and AI-assisted productivity features.

## Features

* JWT Authentication & Authorization
* Role-Based Access Control
* Task Creation, Assignment & Tracking
* Real-Time Notifications with Socket.IO
* Team Collaboration
* Analytics Dashboard
* AI Task Assistant (Google Gemini)
* User Management
* Responsive Design
* Modern UI/UX

## Tech Stack

### Frontend

* Tailwind CSS

### Backend

* Node.js

### Database

* Google Gemini API
